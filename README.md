# 🍽️ Restaurant Sales Analysis | Power BI Dashboard

## 📌 Project Overview
This project involves analyzing restaurant sales data to uncover business insights around revenue trends, order patterns, and item performance. The analysis was built using **Power BI** and **Power Query**, resulting in an interactive multi-page dashboard.

---

## 📊 Dashboard Highlights

| Metric | Value |
|---|---|
| Total Revenue | 160K |
| Total Orders | 5K |
| Total Items Sold | 12K |
| Avg Order Value | 30 |

---

## 🔍 Key Insights

- **March** generated the highest revenue (55K, 34% share), followed closely by February and January
- **Friday** is the busiest day with 24K in daily sales and highest items sold (2.0K on Monday)
- **Peak order hours** are between **16:00–18:00**, with 618 orders at 18:00
- **American and Asian** categories are among the top-performing food segments
- Revenue is fairly evenly distributed across months (~32–34%), indicating consistent demand

## 🧹 Data Preparation

- Removed null and duplicate entries
- Standardized date and time columns
- Created calculated columns for hourly and monthly aggregations
- Categorized dishes by food category for segment analysis

---

## 📈 Dashboard Pages

1. **Sales Timeline** – Revenue and order trends over time
2. **Sales by Items vs Category** – Category-wise performance breakdown
3. **Product Performance** – Top and bottom performing dishes
4. **Insights** – Summary of key business findings

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open with **Power BI Desktop** (free)
3. Use slicers to filter by **Month, Day, Dish, or Category**
